Link: https://docs.nestjs.com/
NestJs
    Framework chạy trên nền Nodejs
    TypeScript: OOP kết hợp lập trình hàm
    Chứa 
        expressjs(framework xây dựng trên nền nodej phổ biến nhất phía server)
        fastify(framework có hiệu suất cao nhất về số lần gửi request/giây)
    Liên lạc với client: angular (react, vue)
    Mã nguồn mở

Install: 
    C1: 
        npm i -g @nestjs/cli
        nest new project-name
    C2: 
        git clone https://github.com/nestjs/typescript-starter.git project
        cd project
        npm install
        npm run start
 
Description: 
    Controller: xử lí yêu cầu và phản hồi tới client
    Provider: nhiều lớp cơ bản có thể là provider: services, repositories, factories, helpers,...
        Ý tưởng provider: 
            có thể inject.
            đối tượng có thể tạo ra các mối quan hiệu
            @Injectable()
    Module: @Module
        Cung cấp metadata sử dụng để tổ chức cấu trúc ứng dụng
    Middleware: 
        Được gọi trước khi route xử lí.
    Exception Filter: xử lí ngoại lệ.
        Khi một ngoại lệ ko bị xử lí nó sẽ bị bắt bởi lớp này
        Tự động gửi một phản hồi thân thiện tới người dùng.
    Pipe: @Injectable
    Guard: @Injectable
        Interface CanActive
    Interceptors: @Injectable
        Interface NestInterceptor

    


Command: 
    nest g mo ten+module: create module
    nest g controller ten+controller: create controller
    nest g service idea
